# Home Assistant Add-on Repository - Claude AI

Repository for Home Assistant Add-ons providing Claude AI Backend integration.

## Add-ons

### Claude AI Backend

Backend API service for Claude AI Assistant Home Assistant integration.

- **Repository**: https://github.com/Bobsilvio/ha-claude-addon-repo
- **Issues**: https://github.com/Bobsilvio/ha-claude/issues
- **Documentation**: https://github.com/Bobsilvio/ha-claude

## Installation

1. Open Home Assistant
2. Go to Settings → Add-ons & backups → Add-ons
3. Click the three-dot menu (⋮) in the top-right
4. Select "Repositories"
5. Add repository: `https://github.com/Bobsilvio/ha-claude-addon-repo`
6. Close the dialog
7. Look for "Claude AI Backend" add-on
8. Click Install

## Supported Architectures

- `x86_64` (Intel/AMD)
- `aarch64` (ARM 64-bit, e.g., RPi 4)
- `armv7` (ARM 32-bit, e.g., RPi 3/2)
