# Changelog

## [1.0.0] - 2026-02-08

### Added
- Initial release
- All-in-one add-on that auto-deploys Claude component + backend API
- Support for Claude Haiku, Sonnet, and Opus models
- 6 services: send_message, execute_automation, execute_script, get_entity_state, call_service, create_automation
- 4 sensors and 1 switch for monitoring
- Multi-language support (IT, EN, ES, FR, DE, NL)

### Features
- Automatic component deployment on add-on startup
- Flask backend API on port 5000
- Configuration UI for model selection and API timeout
- Home Assistant integration with Devices & Services
